Class: BondCards

BondCards

Represents the Bond Cards SDK. It allows developers to make all calls to the APIs that securely store PCI data with a single function.

new BondCards( [live])

Create a BondCards instance.

Parameters:
Name Type Argument Default Description
live boolean <optional>
 true

Set to True to work with live data. False for sandbox data
